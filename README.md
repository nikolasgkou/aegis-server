# aegis-server

Login/game server core for a from-scratch, modern reimplementation of a
Lineage 2-style MMO server — Java, virtual threads, hand-written packet
protocol layer (handshake, framing, crypto).

Part of the `aegis` project. See the other pieces:

- [aegis-infra](https://github.com/nikolasgkou/aegis-infra) — provisioning and GitOps deployment
- [aegis-observability](https://github.com/nikolasgkou/aegis-observability) — dashboards and alerting as code
- [aegis-loadtest](https://github.com/nikolasgkou/aegis-loadtest) — load tests and chaos experiments
- [aegis-admin](https://github.com/nikolasgkou/aegis-admin) — web-based GM/admin dashboard
